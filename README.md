Relay is a real-time communication system designed to enable structured, persistent, and interactive communication between users.

It provides authenticated access, conversation spaces, live message delivery, and long-term message storage using a serverless architecture.

Built with Next.js and powered by Supabase, Relay combines a performant frontend with managed authentication, PostgreSQL storage, and realtime infrastructure.

The system is designed to be scalable, secure, and production-ready.

Relay enables:

* Secure user authentication
* Creation and management of communication spaces
* Real-time message delivery
* Persistent message history
* Structured access control
* Media-ready and extensible messaging architecture
* Responsive, modern user interface
